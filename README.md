## Sobre o Projeto

Este projeto demonstra a implementação da **Metodologia BEM (Block, Element, Modifier)** em um layout de grade de produtos, seguindo as melhores práticas de desenvolvimento CSS escalável e sustentável.

### Objetivo do Exercício

## Refatorar um código HTML/CSS existente para aderir estritamente à nomenclatura BEM, garantindo:
 Separação clara entre blocos, elementos e modificadores
 Código CSS modular e reutilizável
 Manutenção da aparência visual original
 Implementação de boas práticas de acessibilidade e performance
